# Day 1 — Chat Completion API (Ed Donner)

Today I started learning how to use AI models through APIs.

The main idea is that instead of building models from scratch, we can send a request to an API and get a response.

## What I learned

- We can send a prompt (input) to a model
- The model returns a response (output)
- This is called a "chat completion"

## Example idea

Input:
"Explain machine learning in simple terms"

Output:
A simple explanation generated by the AI

## Key takeaway

This makes it possible to build AI apps easily without training models ourselves.
